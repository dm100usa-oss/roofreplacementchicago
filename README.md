# Roof Replacement Chicago

Expert resource for homeowners researching roof replacement in Chicago, IL.

## Stack
- Next.js 14
- TypeScript
- CSS Modules
- Deploy: Vercel (free plan)

## Domain
roofreplacementchicago.com

## Key pages
- / - Homepage with 6 verified contractors
- /insurance-guide - How to get insurance to pay for roof replacement
- /roofing-guide - Complete Chicago roofing guide
- /services/[slug] - 5 service type pages
- /neighborhoods/[slug] - 20 Chicago neighborhood pages
- /es/ - Spanish version of all pages

## Deploy
1. Push to GitHub
2. Connect to Vercel
3. Set domain roofreplacementchicago.com
